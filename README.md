u# Retro tv（基于 Retro Music Player 二次开发）

本项目是一款专为智能电视（Android TV、电视盒子等大屏设备）打造的本地音乐播放器，基于开源项目 [Retro Music Player](https://github.com/h4h13/RetroMusicPlayer) 进行重新开发和大屏适配。

## 🖥 软件简介

- **大屏优化**：界面简洁，布局适配电视大屏幕
- **遥控器支持**：支持遥控器方向键/OK键操作，沙发模式，操作便捷
- **本地播放**：扫描和播放本地音乐文件，无需登陆/联网
- **多种浏览方式**：按专辑、艺人、文件夹、歌曲等管理音频
- **背景播放**：支持边看电视边听音乐
- **歌词显示**：听歌时歌词同步显示
- **主题支持**：多种外观主题可选，适配不同居家风格

## 📺 主要特性

- 适配遥控操作，易上手
- 支持大字体和高对比度模式，远距离清晰可见
- 支持多格式音频文件，如MP3/FLAC/WAV等
- 无广告，无多余权限，纯净播放体验
- 支持自动扫描本地存储音乐

## 🚀 安装方式

### 直接安装

1. 进入 [Releases 页面](https://github.com/soviet1230/-/releases) 下载最新版 APK
2. 拷贝到电视或电视盒子上
3. 允许“未知来源”应用安装
4. 在电视上直接安装并打开应用

### 源码编译

1. 克隆源码
   ```bash
   git clone https://github.com/soviet1230/-.git
   ```
2. 用 Android Studio 打开项目
3. 选择 Android TV 或 TV Box 作为目标设备
4. 运行或生成 APK，即可安装到电视端
